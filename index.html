Elements

Grid

The Acuity grid is a universal grid system combining 12 columns, each divided equally between a gutter of 20px. It should be applied across all devices.

                

    <section class="container">
       <div class="row">
           <div class="cols-[#]>
                // your content goes here.
           </div>
       </div>
   </section>
